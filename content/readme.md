林尽水源，便得一山，山有小口，仿佛若有光。便舍船，从口入。

初极狭，才通人。复行数十步，豁然开朗。

土地平旷，屋舍俨然，有良田美池桑竹之属。阡陌交通，鸡犬相闻。

其中往来种作，男女衣着，悉如外人。黄发垂髫，并怡然自乐。
